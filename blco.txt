Aula 8- Animações: tem o conceito de keyframes(passo a passo da animação, oq vai acontecer com o elemento na linha do keyframe)
ex: @keyframes animacao{                                                 no seletor que for alterar colocar a propriedade desejada 
	from{ background...}                                       ex: { animation-name: animacao
	to{background...}		                            animation-duration: 5s}
}

outras formas de declarar animações:

@keyframes animacao3{
    from{left: 0;} - faz com que a animação nao saia de sua forma estatica brutamente e sim que saia de left 0 ao 200 ja mudando 
    to{left:200px; background-color: green;}                                                                                                   seu formato indicado no keyframe.
}



 
@keyframes animacao2{  porcetamgem em cima do tempo indicado no seletor da propriedade alterada 
    0% {}
    30% {width: 100px;}
    60% {background-color: red;}
    80% {left:100px; background-color: blue;}
    100% {height: 100px;}
}

Propriendades do animation:
*animation-name: animacao2;   - é utilizado para colocar o nome da animation declarado no "keyframes"
*animation-duration: 4s; - indica o tempo da duração da animação 
*animation-delay: 1s; indica o tempo que a animação vai de forma estatica ao seu formato indicado no keyframe
*animation-iteration-count: infinite; - indica que a animação fique de forma infinita a acontecer(pode se usar outros valores: 1, 2, 3 ...)

*animations-timing-function - determina a velocidade com que a animação irá realizar o comando dado no "keyframe"
tipos: linear(mesma velocidade sempre), ease(rapido no começo e devagar no fim), ease-in(devagar no começo e rapido no final)
ease-out, ease-in-out.
ex:
#animation-div4 {
    animation-timing-function: linear;
}

#animation-div5 {
    animation-timing-function: ease;
}

shorthand - animation: nome      tempodeduração     timingfunction      delay      iterationcount

Aula 9 - transition - utilizado em conjunto com o hover para adicionar interações nos elementos .
ex:

.btn{
    display: block;
    position: relative;
    padding: 10px;
    margin-bottom: 25px;
    border-radius: 5px;
    transition: 1s;
}

.btn:hover{
    color: #333;
    background-color: #fff;
}

Aula Extra - Positions

Fluxo do HTML - A posição dos elementos, seguindo a ordem das tags.
Static - Posição padrão dos elementos.
Relative - Posição que podemos modificar as direções.
Absolute - Pode se referenciar pela tela ou por algum elemento.
Fixed - Posição que fixa o elemento na tela.
Sticky - Uma mistura de fixed com relative.
